About

Bot on Turing Test shortly abbreviated bott is a chat bot. More information at http://code.google.com/p/bott

Deployment

  1. Clone the git repo; git clone git@github.com:kailashbuki/bott.git
  2. python run_bott.py

NOTE: There's also an exe for windows machine. Download the compressed file from executable section, uncompress it and double click it to run.
